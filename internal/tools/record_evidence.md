Record a single piece of evidence in the case brain.

Evidence is an **immutable pointer** — a query ID/string, a URL, a file hash, or
a log reference — that backs a specific factual claim. It is not free-form prose.

Use this during the evidence-gathering phase for every fact you intend to state
in the case narrative. The tool returns an evidence ID; you must cite that ID
when you make claims in the case page and when you propose actions. Claims with
no backing evidence are rejected when the case page is written.
